## üìñ About The Project

<img src="Rescuer%20Game/images/desktop.png" alt="Rescuer Game Desktop View" width="100%" />

The **Rescuer Game** is a browser-based game where players take on the role of a rescuer completing missions and saving targets under time pressure.  
The game focuses on smooth gameplay mechanics, simple controls, and visually clear elements for an enjoyable and engaging experience.

This project highlights:
- JavaScript-based game logic  
- Player movement and control handling  
- Interactive UI elements  
- Mission-driven gameplay mechanics  

Ideal for:
- Browser-based game projects  
- JavaScript game development practice  
- Interactive portfolio demos  
- Logic and animation-based experiments  

This is a **front-end only game project**, built for learning, experimentation, and portfolio showcasing.
